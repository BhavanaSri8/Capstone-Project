## 🚀 To Login

Make a POST request to:
```
POST /api/auth/login
```

With this JSON body:
```json
{
  "email": "bhavana@gmail.com",
  "password": "Bhavana@12"
}
```

You'll receive a JWT token to use for authenticated API requests.
